# Demo Questions

Curated questions to showcase GenAnalytics capabilities.

## Questions

1. **Simple metric**: "What was the total payout amount last month?"
2. **Filtered count**: "How many successful payouts happened in the last 7 days?"
3. **Aggregation by category**: "Show payouts by status for the last 30 days"
4. **Time trend**: "Show daily payout count for the last 2 weeks"
5. **Top-N**: "What are the top 10 reward events by count in the last month?"
6. **Schema exploration**: "What tables contain user information?"
7. **Join query**: "Show the total redemption amount by payout status for last month"
